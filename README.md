## 🛠 IDE 사용 주의사항 (IntelliJ IDEA 기준)
### JDK 버전
- **JDK 21 LTS 사용 가능**
- 단, **Java 8 문법만 사용**해야 합니다  
  (교육 목적상 최신 문법 사용 금지)
### 테스트 코드
- `BankServiceTest.java`는 **절대 수정 금지**
- 패키지명은 `bank`로 **main / test 동일**
- 테스트가 인식되지 않을 경우:
  - Maven 프로젝트 여부 확인
  - JUnit5 의존성(pom.xml) 확인
